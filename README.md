# Maintainer Playbook

A practical guide for open source maintainers focused on sustainable maintenance, contributor onboarding, issue triage and pull request review.

## Sections

- Issue triage framework
- Pull request review standards
- Contributor onboarding patterns
- Community health practices
- Governance basics
- Maintenance checklists

## Structure

- `/docs/triage.md`
- `/docs/pull-requests.md`

- `/docs/governance.md`
- `/templates/`

## Goals

This repository aims to provide reusable standards and operational playbooks for maintainers of open source projects.
